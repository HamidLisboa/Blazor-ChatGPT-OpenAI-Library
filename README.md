AI Chat App with .NET & OpenAI SDK

A modern Blazor Web Application that integrates OpenAI's ChatGPT API using the official .NET SDK, enabling real-time AI-powered conversations and intelligent features.

Features
💬 Real-time chat with AI (ChatGPT)
⚡ Streaming responses for enhanced user experience
📦 Integration using the official OpenAI .NET Library
🧠 Prompt handling and response customization

🧰 Tech Stack
Frontend: Blazor (.NET 8)
Backend: ASP.NET Core
Language: C#
AI Integration: OpenAI .NET SDK
Architecture: Client-server (Blazor Server)

What I Learned
Using the official OpenAI .NET SDK in real-world applications
Building AI-powered web apps with Blazor and .NET 8
Implementing streaming responses with the SDK
Designing responsive and interactive AI user interfaces
